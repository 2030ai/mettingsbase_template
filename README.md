# База знаний для встреч

Шаблоны и структура для организации и ведения базы знаний по встречам.

## Структура проекта

- `01_before/` - Шаблоны для подготовки к встрече
- `02_during/` - Шаблоны для ведения заметок во время встречи
- `03_after/` - Шаблоны для последующих действий
- `case/` - Примеры и кейсы
- `база знаний/` - База знаний с записями встреч и контекстом проекта

## Использование

1. Используйте шаблоны из соответствующих директорий для подготовки, проведения и подведения итогов встреч
2. Сохраняйте записи встреч в `база знаний/встречи/` с датированием
3. Обновляйте контекст проекта в `база знаний/контекст проекта.md`
