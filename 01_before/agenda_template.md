## 7) Тайминг (по минутам)

> Правило: если блок закончился — двигаемся дальше, остаток в Parking Lot.

- 00:00–00:02 — Старт: цель, outputs, роли, правила
- 00:02–00:05 — Контекст (только факты, без споров)
- 00:05–00:20 — Решение 1: варианты → критерии → выбор
- 00:20–00:27 — Решение 2 / риски / план
- 00:27–00:30 — Итог: решения + задачи (кто-что-когда) + следующий шаг

## 8) Definition of Done (когда встреча считается успешной)

- [ ] Цель подтверждена в начале
- [ ] Принято минимум 1 решение (или явно зафиксировано, что решения не будет и почему)
- [ ] Зафиксированы Action Items: кто / что / когда / критерий готовности
- [ ] Все TBD вынесены в "Нужно уточнить" с владельцами
- [ ] Понятен следующий шаг и владелец следующего шага
